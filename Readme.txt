ゲームタイトル「オセロ」

＜内容＞
一般的なリバーシをベースにした作品

＜開発環境＞
　Microsoft Visual Studio Community 2017
　DirectX9.0c

＜動作環境＞
　windows8.1 
  windows10

＜開発期間＞
　1月〜6月(5か月)

＜工夫したこと＞
　初めての作品なのでシステムやビジュアルはまだまだですが、
プログラムはクラスの使用やシーン分けなどを利用して工夫できたと思います。

＜苦労したこと＞
企業説明会の時に作品を批評してもらえる場があったので、その時のアドバイスを出来るだけ反映させました。
(特にクラスを使用していなかったので、全体的に使用するように改善しました)

＜ゲームの操作方法＞
　マウスで各箇所をクリック
